GoKit is a modular Go toolkit designed to eliminate boilerplate when building APIs with [Gin](https://github.com/gin-gonic/gin). It packages a structured logger with multiple output targets, JWT access/refresh token management, bcrypt and Argon2id password hashing, and a set of production-ready middlewares — all configurable and ready to drop into your project.

## Getting started

<Steps>
  <Step title="Install GoKit">
    ```bash
    go get github.com/AndresGT/GoKit
    ```
  </Step>
  <Step title="Configure JWT and the logger">
    Call `jwt.Configure` and `logger.InitGlobal` once at application startup before your router is created.
  </Step>
  <Step title="Add middlewares to your Gin router">
    Attach `middleware.DefaultRequestID()`, `logger.GinMiddleware()`, `middleware.DefaultCORS()`, and `middleware.DefaultRecovery()` in order.
  </Step>
  <Step title="Protect routes and read token data">
    Use `middleware.Auth` on protected route groups and `middleware.GetUserID` / `middleware.GetUserRole` inside handlers.
  </Step>
</Steps>
